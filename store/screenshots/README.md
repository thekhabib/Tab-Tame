# Web Store Screenshots

Drop store screenshots here before publishing.

## Specs (Chrome Web Store)

- **Format:** PNG or JPEG
- **Size:** 1280×800 or 640×400
- **Count:** 1–5 (3–4 is sweet spot)
- **No transparency, no rounded corners** — Chrome adds them.

## Suggested shots

1. Popup open with toggle, per-site rule, and stat counter.
2. Options page showing all settings.
3. Tab search popup (Ctrl+Shift+F) with a few tabs filtered.
4. Before / after split: browser with 30 tabs vs. browser with 5 tabs.
